💡 学习建议

1. 先学习基础知识

了解 Godot 的基本操作，包括场景创建、节点操作、TileMap 编辑等。

2. 然后学习玩家系统

掌握角色的创建、控制、状态机设计、动画播放和战斗系统。

3. 接着学习 UI 面板

实现游戏界面，包括 HUD、库存、装备、属性和技能面板。

4. 最后学习物品系统

完成核心玩法，包括物品数据结构、背包管理和物品捡拾。
